 ##Google Books Search

  [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ## DESCRIPTION

  The Google books search app is an application designed to allow the user to search for any books through the google books API, When the user types their desired book in the search bar, and hot submit, the the application populates all the available results for that given book.
